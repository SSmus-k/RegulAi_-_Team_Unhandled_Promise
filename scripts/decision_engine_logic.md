# Decision Engine Logic

The decision engine matches user actions to applicable regulations using modular rules.

## Flow Diagram

```
User Action
   |
   v
Rule Lookup
   |
   v
Regulation Mapping
   |
   v
Checklist Generation
```

## Explanation
- User selects business type and action
- Engine looks up rules for that combination
- Maps to relevant regulations
- Generates step-by-step checklist
- Extensible for new rules/regulations
